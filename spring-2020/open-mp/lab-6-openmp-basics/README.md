# Purpose
Understand basic contepts of OpenMP usage.

# Task
The program must use a sequence of numbers: `{a0…an–1}`. <br/>
It should apply the following formula against given sequence:
```
a0-а1+a2-а3+a4-а5+...
```
<b>Example:</b>
```
Sequence: 1, 2, 3, 4
Expected result: -2. (since 1-2+3-4=-2)
```
